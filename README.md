# Palindrome Checker App

## Overview
The Palindrome Checker App is a simple application designed to determine whether a given string is a palindrome. A palindrome is a word, phrase, number, or other sequences of characters that reads the same forward and backward (ignoring spaces, punctuation, and capitalization).

## Features
- User-friendly interface for inputting text
- Identifies palindromes in real-time
- Highlights non-palindrome strings clearly
- Supports multiple languages

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yash-pratap-create/PalindromeCheckerApp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PalindromeCheckerApp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
1. Open the application in your web browser
2. Enter a string in the input field
3. Observe the real-time results as the application checks if the input is a palindrome
